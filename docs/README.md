# BIOLINK

A high-end, persona-driven AI chatbot and link management platform.

## Features
- **Persona AI**: Chat with a custom AI persona defined in your `.env`.
- **Mobile-First**: Smooth card flip interactions on mobile, side-by-side on desktop.
- **Daily Limits**: 5 questions per day tracked via IndexedDB.
- **Atomic Design**: Modular UI components for maximum scaleability.

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Configure your `.env` with a `VITE_GEMINI_API_KEY`.
4. Run locally: `npm run dev`.
